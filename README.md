# OpenGL Demo

![](res/preview.png)

> 计算机图形血实验

(Core-profile) OpenGL 摄像机+光照演示模型。

## Usage

首先安装GLAD和GLFW，下载`stb_image.h`到`src`文件夹中(参考(Learn OpenGL)[https://learnopengl.com/]上面的教程)。

```
$ mkdir build && cd build
$ cmake .. && make
$ ./main -h
```
